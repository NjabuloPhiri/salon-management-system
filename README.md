## Running the Application

1. **Migrate the database:**
    
    ```bash
    php artisan migrate
    ```
    
2. **Seed the database:**
    
    ```bash
    php artisan db:seed
    ```
    
3. **Run the development server:**
    
    ```bash
    php artisan serve
    ```
